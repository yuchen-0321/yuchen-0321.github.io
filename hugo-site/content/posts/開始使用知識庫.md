# 開始使用個人知識庫

這是我的第一篇知識庫筆記，記錄如何建立和使用這個系統。

## 系統架構

1. **Obsidian** - 本地知識管理
2. **Hugo** - 靜態網站生成
3. **GitHub Pages** - 免費託管

## 使用方式

### 新增筆記
在 Obsidian 中建立新的 Markdown 檔案

### 發布網站
推送到 GitHub 後自動部署
